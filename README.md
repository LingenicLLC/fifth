# Fifth

**Date:** 2026-02-11
**Version:** 0.1 alpha
**Status:** **Alpha version available**

**Fifth** is a modern ColorFORTH compiler that emits SPARK, enabling compilation to bare metal via GNAT. Chuck Moore's minimalist language, running on any platform, with optional verification.

---

## Target Platforms

Via GNAT:

- x86-64 Linux, Windows
- ARM64 Linux, macOS
- Bare metal ARM
- RISC-V
